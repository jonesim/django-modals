Help

Quick guidance for running the demo and adding your first nested modal.

1) Install and configure
  1. Install django-nested-modals and dependencies.
  2. Add bootstrap_modals to INSTALLED_APPS.
  3. Include modal static assets in your base template.
2) Minimal modal example
{% verbatim %} 
{% load modal_tags %}
{% modal_button "Open modal" "my_modal_view" %}
{% endverbatim %}
3) Common issues
  • If nothing opens, verify modal JS/CSS files are loaded on the page.
  • If AJAX requests fail, confirm CSRF middleware and token handling are enabled.
  • If a nested modal closes unexpectedly, check custom JS for Bootstrap modal events.
